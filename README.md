# UK Retail Customer Analytics

## Overview
Analysis of UK retail transactions to identify top 
revenue-generating customers, best-selling products 
and monthly revenue trends.

## Tools Used
- Excel — data cleaning, Pivot Tables and Charts
- SQL — 4 queries using DB Browser for SQLite
- Tableau Public — interactive dashboard

## Dataset
UCI Online Retail II — real UK online retailer transactions
Source: Kaggle — trimmed to 50,000 rows for performance

## Key Findings
- United Kingdom accounts for 85% of total revenue
- White Hanging Heart T-Light Holder is top selling product
- December revenue significantly higher than January
